Этот код демонстрирует основной исследовательский анализ данных с использованием Python с библиотеками pandas, matplotlib и seaborn. Основные этапы включают:

Загрузка данных:

Считывание данных в формате CSV из строки с использованием библиотеки pandas.
Отображение информации о данных:

Показ первых нескольких строк DataFrame.
Вывод информации о DataFrame, включая типы данных и отсутствующие значения.
Расчет основных статистик для числовых столбцов.
Построение графика Countplot:

Использование библиотеки seaborn для создания графика Countplot для переменной 'Severity Level'.
Визуализация частотного распределения каждой категории.
Построение тепловой карты корреляции:

Генерация тепловой карты для визуализации матрицы корреляции числовых переменных.
Аннотации на тепловой карте предоставляют коэффициенты корреляции.
Этот код предназначен для начального исследования данных и визуализации, предоставляя представление о структуре данных, основных статистиках и взаимосвязях между переменными.
